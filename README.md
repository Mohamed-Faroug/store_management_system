# 🏪 نظام إدارة المخزون - مخزن الزينة

نظام شامل ومتقدم لإدارة المخزون والمبيعات والمشتريات، مصمم خصيصاً للمتاجر الصغيرة والمتوسطة، مع واجهة مستخدم عربية متطورة وسهلة الاستخدام.

## 📋 نظرة عامة

يوفر النظام جميع الأدوات التي تحتاجها المتاجر لإدارة:

- الأصناف والمخزون
- المبيعات والفواتير
- المشتريات والموردين
- التقارير والإحصائيات
- صلاحيات المستخدمين
- إعدادات الضرائب والدفع والعملات

## ✨ الميزات الرئيسية

### 🏪 إدارة المخزون
- **إدارة الأصناف**: إضافة، تعديل، حذف
- **تصنيف المنتجات**: فئات متعددة
- **تتبع الكميات**: تنبيهات الكميات المنخفضة
- **تعديل المخزون**: مع تسجيل الأسباب

### 💰 إدارة المبيعات
- **نقطة بيع (POS) متقدمة وسريعة**
- **إدارة الفواتير مع طباعة 58mm و A4**
- **طرق دفع متعددة**: نقدي، بطاقة، شيك
- **حساب الضرائب تلقائيًا**

### 📦 إدارة المشتريات
- **تسجيل المشتريات وتحديث الكميات تلقائيًا**
- **إدارة الموردين وفواتير الشراء**

### 📊 التقارير
- **تقارير يومية / شهرية / سنوية**
- **تقارير المخزون والمبيعات والمشتريات**
- **تصدير إلى Excel**

### ⚙️ الإعدادات
- **إعدادات الضرائب والدفع والعملات**
- **تخصيص واجهة نقطة البيع**
- **معلومات المتجر**

### 👥 إدارة المستخدمين
- **نظام متعدد المستخدمين مع صلاحيات مختلفة**
- **مالك (Owner)**: صلاحيات كاملة للوصول لجميع الميزات المتقدمة
- **مطور (Dev)**: صلاحيات كاملة للوصول لجميع الميزات المتقدمة
- **مدير (Manager)**: إدارة المستخدمين والمخزون والمبيعات
- **كاشير (Clerk)**: استخدام نقطة البيع والمبيعات الأساسية
- **تسجيل دخول آمن وتتبع الأنشطة**

### 📁 إدارة البيانات
- **استيراد / تصدير من وإلى Excel**
- **قوالب جاهزة**
- **نسخ احتياطي يدوي**

## 🚀 التثبيت والتشغيل

### المتطلبات
- **Python 3.7+**
- **Windows 10/11**
- **متصفح حديث**

### الطريقة السهلة (موصى بها)
```bash
git clone https://github.com/Mohamed-Faroug/store_management_system.git
cd store_management_system
```

ثم شغل الملف:
```
INSTALL_AND_RUN.bat
```

### الطريقة السريعة
```bash
pip install -r requirements.txt
python run.py
```

افتح المتصفح واذهب إلى:
- **http://localhost:5000**
- **http://127.0.0.1:5000**

## 🔧 حل المشاكل الشائعة

### مشكلة: خطأ في قاعدة البيانات
إذا ظهرت رسالة خطأ مثل:
```
UnboundLocalError: cannot access local variable 'db' where it is not associated with a value
```

**الحل:**
1. تأكد من أنك تستخدم أحدث إصدار من المشروع
2. احذف ملف `inventory.db` إذا كان موجوداً
3. أعد تشغيل التطبيق - سيتم إنشاء قاعدة البيانات تلقائياً

### مشكلة: لا يمكن الوصول للميزات المتقدمة
**الحل:**
- استخدم أحد هذه المستخدمين:
  - اسم المستخدم: `owner` - كلمة المرور: `owner`
  - اسم المستخدم: `dev` - كلمة المرور: `dev`

### مشكلة: لا يظهر المستخدم `dev` أو `owner`
**الحل:**
1. قم بتشغيل النظام مرة واحدة لإنشاء المستخدمين الافتراضيين
2. تأكد من أن قاعدة البيانات محدثة
3. استخدم أحد المستخدمين المذكورين أعلاه
