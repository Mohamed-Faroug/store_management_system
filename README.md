## 📋 نظرة عامة

نظام إدارة المخزون المتكامل مصمم خصيصاً للمتاجر الصغيرة والمتوسطة، يوفر حلولاً شاملة لإدارة المخزون والمبيعات والمشتريات مع واجهة مستخدم عربية متقدمة.

## ✨ الميزات الرئيسية

### 🏪 إدارة المخزون
- **إدارة الأصناف**: إضافة وتعديل وحذف المنتجات
- **تصنيف المنتجات**: تنظيم المنتجات في فئات مختلفة
- **تتبع الكميات**: مراقبة مستويات المخزون
- **تنبيهات نفاد المخزون**: إشعارات تلقائية عند انخفاض الكمية
- **تعديل المخزون**: تسجيل تعديلات المخزون مع الأسباب

### 💰 إدارة المبيعات
- **نقطة بيع (POS) متقدمة**: واجهة بيع سريعة ومتجاوبة
- **إدارة الفواتير**: إنشاء وعرض وطباعة الفواتير
- **طباعة الإيصالات**: طباعة بحجم 58mm و A4
- **طرق دفع متعددة**: نقدي، بطاقة، شيك
- **حساب الضرائب التلقائي**: حساب الضرائب تلقائياً

### 📦 إدارة المشتريات
- **تسجيل المشتريات**: تسجيل مشتريات من الموردين
- **تتبع الموردين**: إدارة بيانات الموردين
- **تحديث المخزون تلقائياً**: تحديث الكميات عند الشراء
- **فواتير المشتريات**: إنشاء وعرض فواتير المشتريات

### 📊 التقارير والإحصائيات
- **تقارير يومية**: تقارير المبيعات اليومية
- **تقارير شهرية**: إحصائيات شهرية مفصلة
- **تقارير سنوية**: تقارير سنوية شاملة
- **تقارير المخزون**: تقارير حالة المخزون
- **تصدير البيانات**: تصدير التقارير إلى Excel

### ⚙️ الإعدادات المتقدمة
- **إعدادات الضرائب**: تخصيص معدلات الضرائب
- **طرق الدفع**: إدارة طرق الدفع المختلفة
- **العملات**: دعم عملات متعددة
- **إعدادات نقطة البيع**: تخصيص واجهة البيع
- **إعدادات المتجر**: معلومات المتجر والاتصال

### 👥 إدارة المستخدمين
- **نظام متعدد المستخدمين**: دعم عدة مستخدمين
- **صلاحيات مختلفة**: مدير، موظف، محاسب
- **تسجيل الدخول الآمن**: نظام مصادقة آمن
- **تتبع الأنشطة**: تسجيل أنشطة المستخدمين

### 📁 إدارة البيانات

## 🚀 التثبيت والتشغيل

### المتطلبات
- **Python 3.7+**
- **Windows 10/11** (موصى به)
- **متصفح ويب حديث**

### الطريقة السهلة (موصى بها)

1. **تحميل المشروع**
   ```bash
   git clone https://github.com/Mohamed-Faroug/store_management_system.git
   cd store_management_system
   ```

2. **تشغيل ملف التثبيت**
   - انقر نقرتين على `INSTALL_AND_RUN.bat`
   - سيقوم بتثبيت جميع المكتبات المطلوبة
   - سيقوم بإنشاء قاعدة البيانات
   - سيقوم بتشغيل التطبيق

### الطريقة السريعة

1. **تثبيت المكتبات**
   ```bash
   pip install -r requirements.txt
   ```

2. **تشغيل التطبيق**
   ```bash
   # Windows
   START.bat
   
   # أو مباشرة
   python run.py
   ```

### الوصول للتطبيق

افتح المتصفح واذهب إلى:
- **الرابط الرئيسي**: http://localhost:5000
- **الرابط البديل**: http://127.0.0.1:5000

## 🐛 استكشاف الأخطاء

### المشاكل الشائعة

#### خطأ في قاعدة البيانات
```
Error: database is locked
```
**الحل**: تأكد من إغلاق جميع نوافذ التطبيق

#### خطأ في المكتبات
```
ModuleNotFoundError: No module named 'flask'
```
**الحل**: قم بتشغيل `INSTALL_AND_RUN.bat`

#### خطأ في الصلاحيات
```
Permission denied
```
**الحل**: شغل Command Prompt كمدير

#### خطأ في المنفذ
```
Address already in use
```
**الحل**: غير المنفذ في `config.py`

### سجلات الأخطاء
- تحقق من سجل الأخطاء في Console
- راجع ملفات السجل في مجلد `logs/`
- تأكد من صحة إعدادات قاعدة البيانات
