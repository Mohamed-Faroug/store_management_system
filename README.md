# نظام إدارة المخزون والمبيعات - Inventory Management System

نظام شامل لإدارة المخزون والمبيعات والمشتريات مع واجهة مستخدم عربية متقدمة ونظام نقطة بيع متكامل.

## 🌟 المميزات الرئيسية

### 📦 إدارة المخزون
- إدارة الأصناف والفئات
- تتبع الكميات والمخزون
- تنبيهات نفاد المخزون
- تعديل المخزون يدوياً

### 💰 إدارة المبيعات
- نقطة بيع (POS) متقدمة
- إنشاء فواتير مبيعات
- دعم طرق دفع متعددة
- طباعة فواتير (A4 و 58mm)

### 🛒 إدارة المشتريات
- تسجيل مشتريات الموردين
- تتبع تكاليف المشتريات
- إدارة الموردين

### 📊 التقارير والإحصائيات
- تقارير يومية وشهرية وسنوية
- إحصائيات المبيعات والمشتريات
- تحليل الأرباح والخسائر

### ⚙️ الإعدادات المتقدمة
- إعدادات المتجر
- إدارة المستخدمين والأدوار
- إعدادات الضرائب
- طرق الدفع المخصصة
- إعدادات العملات

## 🚀 البدء السريع

### المتطلبات
- Python 3.8+
- SQLite3
- متصفح ويب حديث

### التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/yourusername/inventory-management-system.git
cd inventory-management-system
```

2. **إنشاء بيئة افتراضية**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. **تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

4. **تشغيل التطبيق**
```bash
python run.py
```

5. **الوصول للتطبيق**
افتح المتصفح وانتقل إلى: `http://localhost:5000`

## 🎯 الاستخدام

### 1. إعداد المتجر
1. انتقل إلى **الإعدادات > إعدادات المتجر**
2. أدخل معلومات المتجر (الاسم، العنوان، الهاتف، إلخ)
3. احفظ الإعدادات

### 2. إدارة الأصناف
1. انتقل إلى **الأصناف > إضافة صنف جديد**
2. أدخل تفاصيل الصنف (الاسم، السعر، الكمية، إلخ)
3. اختر الفئة المناسبة
4. احفظ الصنف

### 3. إدارة المبيعات
1. انتقل إلى **المبيعات > نقطة البيع**
2. ابحث عن الأصناف أو تصفح القائمة
3. أضف الأصناف إلى السلة
4. اختر طريقة الدفع
5. أكمل العملية

### 4. طباعة الفواتير
1. من قائمة الفواتير، اضغط على **عرض**
2. اختر نوع الطباعة (A4 أو 58mm)
3. اطبع الفاتورة
