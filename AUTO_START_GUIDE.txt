========================================
    دليل التشغيل التلقائي
    Auto Start Guide
========================================

## نظرة عامة
هذا الدليل يوضح كيفية إعداد نظام إدارة المخزون للتشغيل التلقائي عند بدء النظام.

## الطرق المتاحة

### الطريقة 1: استخدام AUTO_START.bat
1. انسخ ملف AUTO_START.bat إلى مجلد Startup
2. مجلد Startup موجود في:
   - Windows 10/11: C:\Users\[USERNAME]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
3. عند بدء النظام، سيتم تشغيل التطبيق تلقائياً

### الطريقة 2: استخدام START_HIDDEN.vbs
1. انسخ ملف START_HIDDEN.vbs إلى مجلد Startup
2. انسخ ملف AUTO_START.bat إلى نفس المجلد
3. سيتم تشغيل التطبيق في الخلفية بدون إظهار نافذة

### الطريقة 3: إضافة إلى Task Scheduler
1. افتح Task Scheduler
2. اضغط "Create Basic Task"
3. اكتب اسم المهمة: "Store Management System"
4. اختر "When the computer starts"
5. اختر "Start a program"
6. في Program/script: اكتب مسار python.exe
7. في Add arguments: اكتب مسار run.py
8. في Start in: اكتب مسار المشروع

## إعدادات متقدمة

### تشغيل في الخلفية
- استخدم START_HIDDEN.vbs للتشغيل في الخلفية
- لن تظهر نافذة Command Prompt
- التطبيق سيعمل في الخلفية

### تشغيل مع تأخير
- أضف تأخير في بداية AUTO_START.bat:
  timeout /t 30 /nobreak
- سيتم تأخير التشغيل 30 ثانية

### تشغيل مع إعادة المحاولة
- أضف حلقة إعادة المحاولة في AUTO_START.bat
- في حالة فشل التشغيل، سيتم المحاولة مرة أخرى

## استكشاف الأخطاء

### المشاكل الشائعة
1. **التطبيق لا يبدأ تلقائياً**
   - تحقق من وجود الملف في مجلد Startup
   - تحقق من صحة مسار الملفات
   - تحقق من صلاحيات الملفات

2. **خطأ في المسار**
   - تأكد من استخدام المسار الكامل
   - تأكد من وجود Python في PATH
   - تأكد من وجود ملف run.py

3. **خطأ في الصلاحيات**
   - شغل Command Prompt كمدير
   - تحقق من صلاحيات الملفات
   - تحقق من إعدادات Windows

### حلول المشاكل
1. **إعادة تثبيت Python**
   - حمل Python من python.org
   - تأكد من تحديد "Add to PATH"
   - أعد تشغيل النظام

2. **إعادة تثبيت المكتبات**
   - افتح Command Prompt
   - اذهب إلى مجلد المشروع
   - شغل: pip install -r requirements.txt

3. **إعادة إنشاء قاعدة البيانات**
   - احذف ملف inventory.db
   - شغل: python -c "from app.models.database import init_db; init_db()"

## إيقاف التشغيل التلقائي

### الطريقة 1: حذف الملف
- احذف الملف من مجلد Startup
- أعد تشغيل النظام

### الطريقة 2: إعادة تسمية الملف
- أعد تسمية الملف (مثل: AUTO_START.bat.bak)
- أعد تشغيل النظام

### الطريقة 3: إزالة من Task Scheduler
- افتح Task Scheduler
- ابحث عن المهمة
- احذف المهمة
